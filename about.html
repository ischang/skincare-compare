About This Tool This simple tool/wobsite (link to xkcd comic) was written in: vanilla Javascript, some Ajax/JQuery, HTML/CSS and Twitter Bootstrap magic. It's hosted on Github pages, under my own personal Github repository, and all the business logic is done completely client side. Basically, that means nothing you input will get stored (except by your own local machine/browser) -- there's no server on my end or anything that's going to see what you typed or uploaded. This tool is under a Creative Commons License blbh albh albha. If you're interested, you can see everything here at this Github repository. I don't typically write in Javascript, so I can't guarantee that my code is the cleanest or my practices are the best. However, I did put comments above all the methods for my own clarity, since I constantly forget what my code does after coming back to it, months later. This also means that if you're interested in seeing how the code works, you can try to follow along with the comments. If you notice any bugs in the tool, or if something stops working, you can DM me on Reddit @ this username OR contact me below with the debug logs or stacktrace (right click, click on inspect page, and click on Console. If there's any logs, copypaste and send them to me). I won't guarantee that I'll get to fixing something immediately -- after all, I still have a day job and this was created in my spare time. This tool can also work with not just skincare products, but any product that has an ingredient list as long as you provide a CSV with the specified columns. Why I wrote this I wrote this in my spare time for fun. The motivation behind this website is to make my life easier when trying to find skincare products that work for me. I already had a spreadsheet, but it was a pain to try and figure out what particular ingredients was on what list and what wasn't. I have extremely sensitive skin -- I have atopic dermatitis which manifests as mild eczema when triggered, with lots of allergies to nickel, parabens, and other preservatives.
